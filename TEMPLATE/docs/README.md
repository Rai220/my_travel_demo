# docs/

Сюда складывай файлы и подтверждения, чтобы быстро найти офлайн:
- страховки (полисы, контакты ассистанса)
- билеты/маршруты (PDF/скриншоты/PNR)
- подтверждения бронирований (жильё, авто, трансферы)
- важные документы (сканы) и переписку

Совет: называй файлы так, чтобы их было удобно искать:
`YYYY-MM-DD_что_где.pdf` или `тип_поставщик_даты.ext`
