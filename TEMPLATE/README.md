# TEMPLATE — шаблон директории поездки/командировки

Скопируй директорию `TEMPLATE/` и переименуй в формат:
- `YYYY_MM_Название` (например: `2026_04_Париж`, `2026_11_Командировка_Берлин`)

## Что где вести
- `CORE.md`: "суть" поездки — даты, состав, must-have/ограничения, вопросы на уточнение.
- `BOOKINGS.md`: все бронирования и оплаты — статусы, ссылки, номера, условия отмены/депозиты, что еще нужно подтвердить.
- `CHECKLIST.md`: чеклист подготовки — документы, страховка (если требуется), деньги/связь, аптечка, дети (если актуально), перед выездом.
- `MOVEMENTS.md`: перемещения и билеты — сегменты (самолёт/поезд/авто), PNR/номера билетов, пассажиры, багаж, пересадки, план внутренних переездов.
- `PROGRAM.md`: программа мероприятия или план по дням — расписание конференции/кэмпа, либо свободный план активностей для отпуска.
- `docs/`: папка для файлов/сканов/подтверждений (страховки, билеты, ваучеры, переписка, маршруты, важные фото). См. `docs/README.md`.

## Мини-правило
- Держи **фактуру** (номера, ссылки, условия, деньги) в `BOOKINGS.md` и `MOVEMENTS.md`.
- Держи **расписание/программу мероприятия или план по дням** в `PROGRAM.md`.
- Держи **вводные и ограничения** в `CORE.md`, а задачи — в `CHECKLIST.md`.
