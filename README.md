# my_travel_demo

Демонстрационный проект для планирования путешествий и командировок.

**Все данные в этом проекте — вымышленные!** Имена, номера документов, даты, маршруты, бронирования и прочие реквизиты не являются реальными и приведены исключительно для примера.

Проект создан для демонстрации работы с планировщиком путешествий на базе **Claude Code**, **Cursor** и **OpenClaw**.

## Что внутри

- `PERSONS.md` — данные участников поездок (документы, контакты)
- `AGENTS.md` — описание структуры проекта и инструкции для AI-агента
- `TEMPLATE/` — шаблон директории поездки
- `2026_03_Командировка_Сочи_SnowBase/` — пример командировки
- `2026_06_Армения/` — пример семейной поездки за рубеж
- `2026_10_Португалия_Солнечное_затмение/` — пример отпуска с событием

Каждая поездка содержит файлы: `CORE.md`, `BOOKINGS.md`, `CHECKLIST.md`, `MOVEMENTS.md`, `PROGRAM.md` и папку `docs/`.
